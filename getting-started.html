Getting started

Start use quickly and easily

Unpack and we're good to go!

	
simplesocialbuttons/
|__css/
|______sbb.css
|______sbb.min.css
|__less/

Basic template

	
<!DOCTYPE html>
<html>
	<head>
		<meta charset='utf-8' />
		<meta http-equiv="X-UA-Compatible" content="IE=edge" />
		<meta name="viewport" content="width=device-width, initial-scale=1" />

		<title>Simple Social Buttons</title>

		<!-- Minified lib -->
		<link rel="stylesheet" type="text/css" href="css/sbb.min.css" />

	</head>

	<body>

		<!-- Facebook button - Size: normal -->
		<a href="YOUR_LINK_HERE" class='btn facebook'>Like my page</a>

		<!-- Instagram button - Size: big -->
		<a href="YOUR_LINK_HERE" class='btn btn-bg instagram'>Follow my photos</a>

	</body>
</html>
